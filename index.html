COPLANARITY — CRITICALAll work surfaces at exactly 34.5". Verify with 48" straightedge across: saw table → outfeed strips → flip platform top. Shim table saw support shelf with lag-bolt levelers until flush.
PIVOT ALIGNMENTShoulder bolt axis at Z=33.75" exactly (center of 1.5" platform = 33.0 + 0.75). Even 1/8" off and stored position won't be flush with bench top.
BUILD SEQUENCE1) Legs + bottom rails → 2) Mid + top rails → 3) Back panel → 4) Saw support shelf (level to 34.5") → 5) Drop in table saw → 6) Top MDF panels → 7) Flip platform + pivots → 8) Drawers → 9) Casters last.
